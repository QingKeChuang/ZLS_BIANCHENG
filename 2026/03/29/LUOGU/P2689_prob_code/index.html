P2689 东南西北

题目描述

给出起点和终点的坐标，及接下来 TTT 个时刻的顺风方向（东南西北），每个时刻可以选择顺风偏移 111 个单位或者不移动。求到达终点的最少移动步数。

坐标采用平面直角坐标系，xxx 轴正向为东，yyy 轴正向为北。

如果无法偏移至终点，输出 −1-1−1。

输入格式

第一行两个正整数 x1,y1x_1,y_1x1​,y1​，表示起点坐标。

第二行两个正整数 x2,y2x_2,y_2x2​,y2​，表示终点坐标。

第三行一个整数 TTT，表示 TTT 个时刻。

第四至第 T+3T+3T+3 行，每行一个字符，表示顺风方向，即东（E\verb!E!E）南（S\verb!S!S）西（W\verb!W!W）北（N\verb!N!N）的英文单词的首字母。

输出格式

一行一个整数，表示最少移动步数。

我的世界治疗药水效果判断

题目描述

在我的世界中，玩家的生命值上限为20.0点。当生命值较低时，玩家可以使用治疗药水恢复生命值。请根据玩家当前的生命值和药水的恢复量，判断是否需要继续使用药水。

规则

  1. 输入当前生命值（float类型，0.0-20.0）和药水恢复量（float类型，正数）
  2. 使用药水后，生命值不能超过20.0（上限）
  3. 如果使用后生命值仍低于5.0，输出”继续使用药水”
  4. 否则，输出”生命值充足”

输入格式

输入两个float值，用空格分隔：

  • 第一个值：当前生命值（0.0 ≤ 当前生命值 ≤ 20.0）
  • 第二个值：药水恢复量（恢复量 > 0）

输出格式

输出判断结果，只能是以下两个字符串之一：

  • “继续使用药水”
  • “生命值充足”

示例输入与输出

示例1

输入：15.5 3.0
计算过程：

  • 当前生命值15.5 + 药水恢复3.0 = 18.5
  • 18.5 ≤ 20.0（未超过上限）
  • 18.5 ≥ 5.0 → 输出”生命值充足”

输出：生命值充足

示例2

输入：3.0 1.5
计算过程：

  • 当前生命值3.0 + 药水恢复1.5 = 4.5
  • 4.5 < 5.0 → 输出”继续使用药水”

输出：继续使用药水

示例3

输入：19.0 2.5
计算过程：

  • 当前生命值19.0 + 药水恢复2.5 = 21.5 → 超过上限，按20.0计算
  • 20.0 ≥ 5.0 → 输出”生命值充足”

输出：生命值充足

示例4

输入：4.9 0.1
计算过程：

  • 当前生命值4.9 + 药水恢复0.1 = 5.0
  • 5.0 ≥ 5.0 → 输出”生命值充足”

输出：生命值充足

提示

  1. 使用float变量存储当前生命值和药水恢复量
  2. 计算新生命值时，若超过20.0则按20.0处理
  3. 使用if条件判断新生命值是否低于5.0
  4. 注意浮点数比较时的精度问题（本题无需特殊处理）

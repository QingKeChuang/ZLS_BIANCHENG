我的世界生存挑战：怪物攻击后的生命值判断

题目描述

在我的世界中，玩家正在探索一个黑暗的洞穴，突然遭遇了怪物的袭击！请根据怪物类型和玩家当前生命值，判断玩家是否需要使用治疗药水。

怪物类型与伤害值

  • 1: 僵尸 - 造成5.0点伤害
  • 2: 骷髅 - 造成4.5点伤害
  • 3: 蜘蛛 - 造成3.0点伤害

规则

  1. 玩家初始生命值为float类型（范围0.0-20.0）
  2. 根据怪物类型计算受到的伤害
  3. 计算剩余生命值 = 当前生命值 - 伤害值
  4. 如果剩余生命值 < 4.0（即20%生命值），输出”使用治疗药水”
  5. 否则，输出”继续冒险”

输入格式

输入两个值，用空格分隔： - 第一个值：怪物类型（int，1-3） - 第二个值：玩家当前生命值（float，0.0-20.0）

输出格式

输出判断结果，只能是以下两个字符串之一： - “使用治疗药水” - “继续冒险”

示例输入与输出

示例1

输入：1 10.0
计算过程： - 怪物类型1（僵尸），伤害5.0 - 剩余生命值 = 10.0 - 5.0 = 5.0 - 5.0 >= 4.0，输出”继续冒险”

输出：继续冒险

示例2

输入：2 6.0
计算过程： - 怪物类型2（骷髅），伤害4.5 - 剩余生命值 = 6.0 - 4.5 = 1.5 - 1.5 < 4.0，输出”使用治疗药水”

输出：使用治疗药水

示例3

输入：3 3.5
计算过程： - 怪物类型3（蜘蛛），伤害3.0 - 剩余生命值 = 3.5 - 3.0 = 0.5 - 0.5 < 4.0，输出”使用治疗药水”

输出：使用治疗药水

提示

  1. 使用int变量存储怪物类型，float变量存储生命值和伤害值
  2. 使用if-else语句判断怪物类型，确定伤害值
  3. 计算剩余生命值后，用if判断是否需要使用药水
  4. 注意浮点数比较时的精度问题（本题无需考虑，直接比较即可）

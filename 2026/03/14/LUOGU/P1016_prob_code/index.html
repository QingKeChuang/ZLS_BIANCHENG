题目背景

None

题目描述

一个旅行家想驾驶汽车以最小的费用从一个城市到另一个城市（假设出发时油箱是空的）。给定两个城市之间的距离 、汽车油箱的容量 （以升为单位）、每升汽油能行驶的距离 、出发点每升汽油价格 和沿途油站数 ，油站 离出发点的距离 、油站 每升汽油价格 ，你需要求出最小的费用。

输入格式

第一行，四个实数 和一个整数 ，含义见题目描述。

输出格式

如果无法到达目的地，输出 No Solution。

P1051 [NOIP 2005 提高组] 谁拿了最多奖学金

题目背景

None

题目描述

某校的惯例是在每学期的期末考试之后发放奖学金。发放的奖学金共有五种，获取的条件各自不同：

  1. 院士奖学金，每人 元，期末平均成绩高于 分（），并且在本学期内发表 篇或 篇以上论文的学生均可获得；
  2. 五四奖学金，每人 元，期末平均成绩高于 分（），并且班级评议成绩高于 分（）的学生均可获得；
  3. 成绩优秀奖，每人 元，期末平均成绩高于 分（）的学生均可获得；
  4. 西部奖学金，每人 元，期末平均成绩高于 分（）的西部省份学生均可获得；
  5. 班级贡献奖，每人 元，班级评议成绩高于 分（）的学生干部均可获得；

只要符合条件就可以得奖，每项奖学金的获奖人数没有限制，每名学生也可以同时获得多项奖学金。例如姚林的期末平均成绩是 分，班级评议成绩 分，同时他还是一位学生干部，那么他可以同时获得五四奖学金和班级贡献奖，奖金总数是 元。

现在给出若干学生的相关数据，请计算哪些同学获得的奖金总数最高（假设总有同学能满足获得奖学金的条件）。

第一行是 个整数 ，表示学生的总数。

接下来的 行每行是一位学生的数据，从左向右依次是姓名，期末平均成绩，班级评议成绩，是否是学生干部，是否是西部省份学生，以及发表的论文数。姓名是由大小写英文字母组成的长度不超过 的字符串（不含空格）；期末平均成绩和班级评议成绩都是 到 之间的整数（包括 和 ）；是否是学生干部和是否是西部省份学生分别用 个字符表示， 表示是， 表示不是；发表的论文数是 到 的整数（包括 和 ）。每两个相邻数据项之间用一个空格分隔。

输出格式

  • 第 行是获得最多奖金的学生的姓名。如果有两位或两位以上的学生获得的奖金最多，输出他们之中在输入文件中出现最早的学生的姓名。
  • 第 行是这名学生获得的奖金总数。
  • 第 行是这 个学生获得的奖学金的总数。

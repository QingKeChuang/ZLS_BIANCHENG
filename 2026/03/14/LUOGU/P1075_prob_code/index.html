题目背景

题目描述

已知正整数 nnn 是两个不同的质数的乘积，试求出两者中较大的那个质数。

输入格式

输入一个正整数 nnn。

输出格式

输出一个正整数 ppp，即较大的那个质数。

提示

1≤n≤2×1091 \le n\le 2\times 10^91≤n≤2×109
